# Data Science Career Analysis: The Applied Scientist Premium

## 🎯 Project Overview
This project investigates the compensation landscape for data professionals using 2024-2025 global salary data. The goal was to identify high-value career paths for entry-level talent and analyze how work settings and company sizes impact total compensation in the current AI-driven market.

## 🚀 Key Insights
- **The Applied Scientist Lead:** Identified "Applied Scientist" as the top-earning entry-level role, with average compensation reaching ~$190,000.
- **Remote Work Premium:** Discovered that Remote roles for specialized AI positions carry a slight salary premium over In-Person roles, challenging traditional office-centric pay myths.
- **Market Competition:** Medium-sized companies (50-500 employees) are currently outbidding Large enterprises for elite talent by approximately 1.2%, suggesting a highly competitive mid-market.

## 🛠️ Technical Implementation
- **Data Integrity:** Performed statistical cleaning to remove "Hybrid" outliers with insufficient sample sizes (N=1), ensuring the final analysis was based on significant data points.
- **Libraries:** Python, Pandas, Matplotlib, Seaborn.
- **Environment:** Developed and tested within PyCharm.
